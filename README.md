# Budget System (FinanceSys_Test)

A simple **desktop budgeting application** built with **C# .NET Windows Forms**. 

It features user authentication via a login form and basic CRUD operations for financial records. This is a test/practice project demonstrating Windows Forms development with clean UI and utility functions.

## Features

- **Login System**  
  Secure user authentication before accessing the application.

- **Dashboard**  
  Overview of financial summary (likely balances, totals, charts, or key metrics).

- **Incomes Management**  
  Add, view, edit, and delete income records.

- **Expenses Management**  
  Add, view, edit, and delete expense records.

- **Shared Functions**  
  Reusable utility methods for common operations (e.g., data handling, calculations).
